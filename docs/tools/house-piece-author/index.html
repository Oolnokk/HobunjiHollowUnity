1. Piece + footprint grid

The 2D grid is the modular footprint contract. Blue cells drive the house shell. Green cells generate roofless entryway tunnels with a door at the far end; gray cells generate chimneys; brown cells generate porches; tan cells generate porch stairs; dark posts generate fences/railings with auto-connectors. The white dashed box is only a measuring/debug guide.

2. Generate / edit base shape

Generation uses the selected preset. Blue painted cells are the main shell; extension tiles are generated as attached add-ons after the main body so they can stay rectangular/roofless even when the Highland body is a frustum.

3. Face tools

Selected face: none

Face edits are vertex-level and remain part of the main shape. Numeric Scale U/V is tile-relative: 0.9 means 90% of the selected face's nearest grid-aligned span, not 90% of its current size. GLBs are separate detail assets and keep their own transforms.
